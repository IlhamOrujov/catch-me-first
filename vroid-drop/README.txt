♡ VRoid drop folder ♡

Export a .vrm from VRoid Studio into THIS folder (File → Export → VRM),
and the game's 🎨 Character Studio will detect it automatically —
click it to load her, or turn on "VRoid Live" to auto-load the newest export.

Tip: point the server straight at VRoid Studio's export folder so you don't
even have to copy files:
    VROID_WATCH="/path/to/your/vroid/exports" python3 server.py
